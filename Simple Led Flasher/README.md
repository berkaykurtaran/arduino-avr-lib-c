# Simple Led Flasher
  - connect LED anode to pin 13 on Arduino Uno Board (If you plan to use another pin you should use a resistor around 220 ohm on circuit)
  - connect LED cathode to GND

## Assemble
```bash
make hex
```
## Flash
```bash
make flash
```
## To Disassemble Elf
```bash
make disassemble
```
